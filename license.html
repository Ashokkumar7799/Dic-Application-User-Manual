Startup Screen

License & Startup

The first screen you see when launching the DIC Application. You cannot access any other part of the software until your license is validated. This section explains every element on the startup screen and walks you through the activation process.

Figure 1.1 — License & Startup screen on first launch

1.1   Header

At the very top of the screen, the application displays its identity information. No user action is needed in this area.

  • InsituMicron logo / title image — confirms which software you are running.
  • "Digital Image Correlation Framework" — the full product name.
  • "Professional Edition" — the software edition label.

1.2   License Status Banner

Directly below the header, a color-coded banner shows your current license state at a glance.

Banner ColorMessage DisplayedMeaning
Yellow"License Required — Please activate your license to continue"No valid license is saved on this machine. You must enter and activate a key before you can proceed.
Green"✓ Licensed to: [Your Name] | Expires: [Date] ([N] days remaining)"A valid license is already stored on this machine. The Continue button is automatically enabled.
Figure 1.2a — Yellow banner indicating no license found
Figure 1.2b — Green banner showing an active license
If the banner is green, the application will automatically enable the Continue button. You can proceed directly without re-entering any key.

1.3   Terms and Conditions

A scrollable text area displays the full Software License Agreement. You must read and accept this agreement before the Activate button becomes clickable.

ElementPurpose
Terms and Conditions (scrollable box)Displays the full license agreement text. Scroll to read the entire document before accepting.
Checkbox — "I have read and agree to the Terms and Conditions"You must tick this checkbox to enable the Activate button. Leaving it unticked keeps the button grayed out and non-functional.
Figure 1.3 — Terms and Conditions section with the agreement checkbox
You must scroll through the Terms and Conditions and tick the checkbox before the Activate button will respond to clicks. This is enforced by the application.

1.4   Machine ID & License Key

This section is used during first-time activation. It contains three key elements that work together to bind your license to this specific computer.

Figure 1.4 — Machine ID, Copy button, and License Key entry
ElementPurposeAction Required
Your Machine ID (read-only text box)A unique identifier generated from your PC's hardware. This code is tied permanently to this specific computer.Do not edit this field. Simply read the value or use the Copy button.
Copy (button)Copies your Machine ID to the Windows clipboard so you can paste it into an email or form.Click Copy, then send the Machine ID to your software vendor. They use it to generate a license key that only works on this computer.
Enter License Key (large text box)The field where you paste the license key received from your vendor.Paste your license key here exactly as provided — including all hyphens and characters. Do not add or remove spaces.
Your license key is hardware-locked. It will only activate on the computer whose Machine ID was submitted to the vendor. Attempting to use the same key on a different machine will result in an "Invalid license key" error.

1.5   Activation Button

Figure 1.5 — The Activate button and its states
ButtonState / ColorFunction
Activate License & ContinueGrey (disabled) until the Terms checkbox is tickedReads the key in the License Key box, validates it against the current Machine ID, and — if valid — opens the main application workspace.
✓ Continue to ApplicationGreen — only shown when a valid license is already storedSkips key entry entirely and opens the main application directly.

After clicking Activate License & Continue:

  • If successful: The status banner turns green and displays your name and expiry date. The application launches automatically after approximately one second.
  • If it fails: A red error message appears below the button — "✗ Invalid license key". Verify you copied the full key without leading or trailing spaces.

First-Time Activation — Step by Step

Scroll through the Terms and Conditions text box. Read the agreement before proceeding.

Tick the "I have read and agree to the Terms and Conditions" checkbox. The Activate button will become clickable.

Click the Copy button to copy your Machine ID to the clipboard.

Send the Machine ID to your InsituMicron software vendor (by email or the method specified in your purchase agreement). They will generate a license key for your machine.

When you receive your license key, paste it into the Enter License Key text box. Ensure the entire key is captured.

Click Activate License & Continue. The application will validate the key and open.

Once activated, the license is saved locally on your machine. On subsequent launches, the application will detect the stored license and show the green banner — you will not need to re-enter the key.
